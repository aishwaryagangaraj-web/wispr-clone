# Wispr Clone

A real-time voice-to-text desktop application built using Tauri and Deepgram.

## Features
- Desktop app using Tauri
- Real-time speech to text
- Start / Stop recording
- Clean and minimal UI

## Tech Stack
- HTML, CSS, TypeScript
- Tauri
- Deepgram API

## How to Run
npm install  
npm run tauri dev

## Notes
Deepgram API key is required to run the project.
API key is not committed for security reasons.
